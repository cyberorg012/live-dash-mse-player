# dash mse player

## Running

* npm install
* bower install
* gulp serve
* browse to http://localhost:7000/index.html


## Chrome

* OS X: supported
* Windows: untested
* Linux: untested
* Android: untested


## Safari

* OS X: supported
* iOS: untested (iOS 8 supports MSE, so possibly)


## Firefox

To enable MSE:

1. update
2. browse to "about:config"
3. set media.mediasource.enabled to true
4. set media.mediasource.youtubeonly to false

* OS X: supported
* Windows: untested
* Linux: untested
* Android: untested
